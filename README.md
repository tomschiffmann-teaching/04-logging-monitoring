# Logging & Monitoring — Course Exercises

Two hands-on exercises. Only **basic Python** and **Docker / Docker Compose** are required.

```
01-logging/      → Exercise 1: from print() to real logging, then debug-by-logging
02-monitoring/   → Exercise 2: Docker container logs, then metrics + dashboards
```

## 👉 New to the terminal / Python? Read `SETUP.md` first
**No Python knowledge is required.** `SETUP.md` shows you, step by step, how to open a
terminal, move into a folder (`cd`), run a Python file (`python3 file.py`), add a log line,
and start the Docker exercises. Every command you need tonight is listed there.

## What you need installed
- Python 3.10+ (`python3 --version`)
- Docker + Docker Compose (`docker --version`, `docker compose version`)

No `pip install` is needed for Exercise 1 — Python's `logging` is built in.
(Install help is in `SETUP.md`.)
